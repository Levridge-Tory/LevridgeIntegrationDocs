    D365 F&O to D365 CRM

    D365 F&O to D365 CRM

    Setup

    To integrate from D365 F&O to D365 CRM you will need to: - Configure Event Endpoint in F&O - configure Levridge Entity Events - Create an application ID for the integration framework to authenticate to D365 CRM - Create an application user in D365 CRM and assign the proper role(s) - Create an Azure Service bus topic - Create a subscription on the topic above

    Configuration

    In the appsettings.json you will need to define the SourceConfig and TargetConfig nodes as follows:

    "SourceConfig": {
    "ServiceBusConfigName": [section name with service bus topic],
    "ODataConfigName": [section name with F&O data configuration],
    "SystemName": "DynamicsAX",
    "Direction": "Source"
},
"TargetConfig": {
    "ODataConfigName": [section name with CRM data configuration],
    "CDSConfigName": [section name with CDS data configuration],
    "SystemName": "DynamicsCRM",
    "Direction": "Target"
}

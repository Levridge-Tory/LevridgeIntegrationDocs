    Configure Appsettings Files

    Configure the appsettings.json file in the following default locations: C:\Program Files (x86)\Levridge\LevridgeScaleHouse\Servers\LevScaleClient C:\Program Files (x86)\Levridge\LevridgeScaleHouse\Servers\LevScaleAPI C:\Program Files (x86)\Levridge\LevridgeScaleHouse\Services\AxToScaleIntegration C:\Program Files (x86)\Levridge\LevridgeScaleHouse\Services\HardwareInterface
